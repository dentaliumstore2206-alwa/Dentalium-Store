# Dentalium-Store - Medical E-Commerce Platform

> Platform e-commerce untuk alat medis, peralatan kedokteran, dan obat-obatan

## Setup Cepat (2 Menit)

### Deploy ke Vercel (RECOMMENDED)
1. Visit https://vercel.com
2. Click 'New Project'
3. Import from GitHub: https://github.com/anonymooooos-990/Dentalium-Store
4. Click Deploy
5. Done! Your site is live!

### Local Setup
```bash
git clone https://github.com/anonymooooos-990/Dentalium-Store.git
cd Dentalium-Store
npm install
npm run dev
```

## Features
- 125+ Medical Products across 5 categories (Instruments, Consumables, Diagnostics, Protective, Medications)
- Search & Filter by category and name
- Shopping Cart with registration requirement
- WhatsApp integration for orders
- Live chat support
- Responsive Design with Red-White-Light Blue theme
- Company information and contact pages
- Fast Performance with Next.js

## Color Theme
- Red: #ff0000
- White: #ffffff
- Light Blue: #add8e6
